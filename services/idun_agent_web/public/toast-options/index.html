The single biggest tradeoff

A is quietest but relies on users noticing a thin stripe — poor for errors that demand attention. B is immediately readable but the color tint can clash with busy backgrounds. C carries the most information but takes more vertical space on mobile and requires callers to provide both a title and a body string.

All three fix the headline bugs: responsive width, absolute-positioned X button, no anchor-center, no !important, icon + text wrapping correctly.
