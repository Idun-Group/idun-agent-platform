Tradeoffs

ProsMost self-explanatory at a glance. "Workspace · Acme Corp" reads like natural language — zero learning curve. Works well at all text sizes. Prefix creates visual rhythm between the two selectors.
ConsWider than the current design. At mobile the text prefix is hidden and falls back to icons (see 375 px preview), so the benefit is lost at narrow viewports. May feel verbose once users are familiar.
Best forFirst-time users or products where workspace confusion is a reported issue. Strong choice if analytics show users frequently selecting the wrong project.
