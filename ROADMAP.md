# Idun Agent Platform Roadmap

The platform is under active development and this document tracks major milestones.

## Completed

- [x] MVP released on PyPI as `idun-agent-engine`
- [x] Local and self-hosted deployment with Docker
- [x] Adapters for LangGraph and ADK
- [x] AG-UI compatible CopilotKit API endpoint
- [x] MCP server support
- [x] Guardrails using Guardrails AI
- [x] Observability integrations (Langfuse, LangSmith, Phoenix, GCP Trace)
- [x] SSO access to Manager UI


## In progress / planned

- [ ] More agent frameworks and MCP integrations
- [ ] Environment management (DEV / STG / PRD)
- [ ] Multimodal endpoint support
- [ ] A2A (Agent-to-Agent protocol): extended configuration for A2A agents
- [ ] Ready-to-use agent templates (Deep Search, Translation, etc.)
- [ ] Agent UI templates
- [ ] RBAC SSO for agent access
- [ ] Secret manager integrations with cloud providers
- [ ] Dashboarding and logs within UI
- [ ] Expanded observability and evaluation integrations (feedback)
- [ ] Terraform deployment template
- [ ] Helm / Kubernetes deployment template
- [ ] Agent Gateway
- [ ] Ready-to-use MCP tools
- [ ] Idun Cloud managed offering

This roadmap is intentionally high level and will be updated as features are delivered.
