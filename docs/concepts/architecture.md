# Architecture

## System Overview

The Idun Agent Platform architecture consists of 3 components working together.

<!-- Architecture diagram -->

## Components

### Engine

The execution engine handles agent runtime, lifecycle management, and framework integration.

### Manager

The manager provides API and UI interfaces for configuring, deploying, and monitoring agents.

### Schema

The schema system defines agent configurations, capabilities, and interfaces in a standardized format.

## Integration Points

The platform integrates with various agent frameworks, observability tools, and deployment infrastructure.
