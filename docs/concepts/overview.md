# Overview

## Introduction

Idun Agent Platform is a comprehensive framework designed to simplify the development and deployment of AI agents across multiple frameworks.

## Core Concepts

The platform provides a unified approach to agent management, allowing developers to work with different agent frameworks through a consistent interface.

## Key Components

The platform consists of three main components:
- Idun Agent Engine: The engine responsible for configuring, managing and serving an agent.
- Idun Agent Manager: Service that handles Agent management, Auth, Infra and so on.
- Idun Agent Schema: Schema definitions for the manager and the engine.
