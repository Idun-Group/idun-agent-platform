# Configuration Reference

## Overview

Complete reference for all configuration options available in config.yaml files.

## Top-Level Fields

### name

Agent identifier used for reference and logging.

### framework

Specifies which agent framework to use for execution.

### version

Configuration schema version for compatibility tracking.

## Framework-Specific Options

Configuration options that vary based on the selected framework.

## Runtime Parameters

Settings that control agent execution behavior and resource allocation.

## Security Settings

Authentication, authorization, and encryption configuration options.

## Advanced Configuration

Optional advanced settings for fine-tuning agent behavior and performance.
