# Environment Variables

## Overview

Environment variables used to configure Idun Agent Platform components.

## Core Variables

### IDUN_API_KEY

API key for authentication with the Idun Manager.

### IDUN_MANAGER_URL

URL of the Idun Manager instance to connect to.

### IDUN_LOG_LEVEL

Logging verbosity level for debugging and monitoring.

## Framework Variables

Environment variables specific to individual agent frameworks.

## Database Configuration

Variables for configuring database connections and credentials.

## Observability Settings

Environment variables for observability tool integration.

## Security Configuration

Variables for managing secrets, certificates, and security settings.

## Development Mode

Variables that enable development and debugging features.
