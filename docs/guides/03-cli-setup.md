# Guide 3: CLI Setup

## Overview

This guide covers using the CLI to manage agent configurations from the manager or from files.

## Get Configuration from Manager

Retrieve agent configurations stored in the manager using CLI commands.

```bash
idun config get my-agent
```

## Get Configuration from File

Load and validate configuration from local YAML files.

```bash
idun config load -f config.yaml
```
